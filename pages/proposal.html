SERVICIOS OFRECIDOS

DISEÑO WEB RESPONSIVE

Diseñamos, desarrollamos e implementamos sitios web con las últimas tecnologías y cumpliendo todos los estándares y recomendaciones de Google. Trabajamos para que tu sitio web obtenga la mejor performance para su correcta indexación en los buscadores

DESARROLLO

El proyecto se presentara como un boceto donde el cliente podrá ver todas las opciones y donde se aplicaran los cambios hasta llegar al objetivo final. Se realizara un diseño a medida donde se aplicaran distintas tecnicas de diseño para que tenga posibilidades de ser adaptado a tus necesidades. Nuestra propuesta es original y la vamos ajustando hasta obtener un sitio web personalizado y único.

SEO & MARKETING

Estrategia de optimización holísticas, con foco en la calidad y la experiencia del usuario, que contemplan todos los aspectos relacionados con el sitio web y su contexto. Tenemos como objetivo maximizar la visibilidad en Google, para que logren conectar con su público objetivo.

WEB APP

El cliente necesita el diseño de una aplicacion dentro del sitio web que interactue con el Hardware que comercializa. Para dicha aplicacion se deberá trabajar en conjunto con el equipo de desarrollo de hardware de MUTED INSTRUMENTS. Se Se utilizaran como base React js, python y como parte de la comunicacion el Framework WebMidiJS que incluye soporte nativo en Chrome,Edge. Definiremos la arquitectura tecnológica más eficiente y conveniente sobre la cual viabilizar la integración de los procesos.
